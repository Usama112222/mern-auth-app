# 🔐 MERN Authentication System

A full-stack authentication system built using the MERN stack (MongoDB, Express, React, Node.js).  
This project includes user registration, login, JWT authentication, and protected routes.

---

## 🚀 Features

- User Signup & Login
- JWT Authentication
- Protected Routes (Frontend + Backend)
- Password Hashing (bcrypt)
- MongoDB Database
- REST API (Express + Node.js)
- React Frontend (SPA)
- Axios API integration
- Logout functionality

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM
- Context API (or Redux if used)

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (JSON Web Token)
- bcrypt.js
- dotenv
